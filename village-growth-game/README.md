# 🏡 마을 성장 시뮬레이션 (Village Growth Sim)

React + Vite + HTML5 Canvas로 만든 **2D 탑다운 마을 성장 게임**입니다.
스타듀밸리·애니멀크로싱·Forager에서 영감을 받아, 캐릭터를 직접 움직이며
도구로 자원을 모으고, 농사·어업·축산으로 마을을 키우고, 교역하며 직위를 올립니다.
모든 그래픽은 코드로 그린 통일된 픽셀아트 스타일이며, 진행은 자동 저장됩니다.

## 핵심 특징

- **🎮 2D 탑다운 월드** — 캔버스에 그려지는 큰 타일 맵을 캐릭터로 자유롭게 탐험. 카메라가 따라오고 휠/버튼으로 확대·축소.
- **🚶 부드러운 이동** — 방향키/WASD(또는 모바일 조이스틱)로 이동, 바라보는 방향이 바뀜.
- **🛠️ 도구** — 🪓 도끼(벌목→목재), ⛏️ 곡괭이(채굴→돌), 🎣 낚싯대(물가 낚시), 🌱 씨앗(밭에 파종/수확). 핫바(1~4)로 전환.
- **🏗️ 건물 배치** — 집·우물·가로등·울타리·화단을 목재/돌을 소비해 원하는 곳에 직접 설치. 마을이 실제로 커집니다.
- **🌅 낮과 밤** — 시간이 흐르며 조명이 변하고, 밤에는 가로등과 창문이 빛납니다.
- **🍂 계절** — 봄·여름·가을·겨울에 따라 맵 전체 색감이 변합니다.
- **🧑‍🌾 NPC** — 마을 주민들이 목적지를 따라 돌아다닙니다.
- **🏠 시설 진입** — 마을회관·시장·낚시터·항구·축사·연구소에 들어가면 해당 관리 패널이 열립니다.
- **📈 경영 시스템** — 농업(5작물)·어업(희귀어)·축산(가축/축산물)·시장(가격변동)·교역(인근 마을)·직위(평민~국왕)·연구(5분야)·주민(행복도/세금/식량).
- **🪟 미니멀 UI** — 화면 대부분이 게임 월드. 자원/정보 패널은 좌우에서 접고 펼치는 드로어.
- **📱 반응형** — PC(키보드+마우스+휠)와 모바일(조이스틱+터치)을 모두 지원.

## 조작법

| 입력 | 동작 |
|------|------|
| 방향키 / WASD / 조이스틱 | 이동 |
| 스페이스 / E / ✋ 버튼 / 클릭 | 정면 타일에 행동(도구 사용·진입) |
| 1~4 | 도구 선택 |
| 마우스 휠 / ➕➖ | 확대·축소 |
| ⏸ ▶ ⏩ | 시간 정지/진행/가속 |
| 📊 / 🏛️ (가장자리 탭) | 자원·정보 패널 열기 |

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```

## 구조

```
src/
  game/
    world/
      worldgen.js   # 맵·자연물·건물·NPC 절차적 생성
      draw.js       # 캔버스 픽셀아트 렌더링
      palette.js    # 계절 색감 + 낮/밤 조명
    constants.js crops.js fishing.js livestock.js
    buildings.js research.js goods.js          # 경영 시스템 데이터
    engine.js reducer.js initialState.js storage.js util.js
  hooks/
    useGame.js      # 시뮬레이션 상태(reducer) + 저장/불러오기
    useWorld.js     # 게임 루프·입력·카메라/줌·낮밤·NPC·도구
  components/
    GameCanvas.jsx  # 캔버스 + 미니멀 오버레이 UI(핫바/드로어/조이스틱/모달)
    ResourcePanel.jsx InfoPanel.jsx
    panels/         # 낚시·축산·건설·연구·시장·교역 관리 패널
    world/Modal.jsx
  App.jsx
```
