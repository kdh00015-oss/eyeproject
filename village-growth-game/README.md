# 🏡 마을 성장 시뮬레이션 (Village Growth Simulation)

React + Vite로 만든 2D 마을 성장 시뮬레이션 게임입니다.
농업·어업·축산으로 자원을 생산하고, 마을을 키우며, 다른 마을과 교역하고,
평민에서 국왕까지 직위를 올려가는 종합 경영 시뮬레이션입니다.
진행 상황은 자동 저장되어 언제든 **이어하기**가 가능합니다.

## 주요 시스템

- **⏱️ 시간/계절/날씨** — 봄·여름·가을·겨울이 순환하고, 맑음·비·폭풍·가뭄 날씨가 생산량에 영향을 줍니다. 재생 속도(정지/1·2·3배속) 조절 가능.
- **🌾 농업** — 밀·감자·옥수수·딸기·포도. 작물별 성장일·수확량·판매가가 다르며, 계절·날씨·연구에 따라 수확량이 달라집니다. 파종 → 성장 → 수확.
- **🎣 어업** — 강·호수·바다 낚시터. 낚시터별 물고기와 가격이 다르고, 확률 기반 희귀 물고기가 등장합니다. (바다는 항구 필요)
- **🐮 축산** — 닭·돼지·소·양. 매일 사료를 소비하고 달걀·송로버섯·우유·양털을 생산합니다.
- **🏗️ 건설** — 집·창고·시장·항구·연구소·학교. 건물마다 효과가 있고 레벨 업그레이드가 가능합니다.
- **👥 주민** — 인구·행복도·세금·식량 소비. 행복도가 높으면 인구가 늘고, 낮으면 줄어듭니다.
- **💹 시장** — 작물·생선·축산물 가격이 매일 변동합니다. 비쌀 때 판매하세요.
- **🤝 교역** — 인근 마을을 탐험해 발견하고, 교역로를 개설해 매일 수익과 영향력을 얻습니다.
- **🎖️ 직위** — 평민 → 작업반장 → 마을 관리자 → 촌장 → 시장 → 영주 → 총독 → 국왕. 조건을 충족하면 자동 승급됩니다.
- **🔬 연구** — 농업·어업·축산·상업·행정 5개 분야. 레벨이 오를수록 생산량과 수익이 증가합니다. (연구소 필요)

## 저장 / 이어하기

- 진행 상황은 브라우저 `localStorage`에 **자동 저장**되어 새로고침해도 유지됩니다.
- 상단 `💾 저장` 버튼으로 즉시 저장, `🌱 새 게임` 버튼으로 처음부터 시작합니다.

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```

## 프로젝트 구조

```
src/
  game/                # 게임 로직 (UI와 분리)
    constants.js       # 계절·날씨·직위·마을 등급·핵심 수치
    crops.js           # 농업 작물 정의
    fishing.js         # 어업 낚시터·물고기 정의
    livestock.js       # 축산 가축·축산물 정의
    buildings.js       # 건물 정의·비용 계산
    research.js         # 기술 연구 정의
    goods.js           # 판매 재화 통합 카탈로그
    initialState.js    # 새 게임 초기 상태
    engine.js          # 파생 통계 + 하루 진행(advanceDay)
    reducer.js         # 플레이어 행동 처리
    storage.js         # localStorage 저장/불러오기
    util.js            # 공용 유틸
  hooks/
    useGame.js         # 상태·시간 루프·행동 디스패치 통합 훅
  components/
    TopBar.jsx         # 상단: 날짜/계절/날씨/직위/등급
    ResourcePanel.jsx  # 좌측: 자원 현황
    ActionPanel.jsx    # 중앙: 탭 컨테이너
    InfoPanel.jsx      # 우측: 건물/주민/마을 정보
    LogPanel.jsx       # 하단: 진행 로그
    panels/            # 농사·낚시·축산·건설·연구·시장·교역 탭
  App.jsx              # 전체 레이아웃 조립
```
