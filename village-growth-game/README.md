# 🏡 마을 키우기 (Village Growth Game)

React + Vite로 만든 마을 키우기 게임입니다.
**방치형(클릭) + 자원 관리 시뮬 + 타일 배치**를 한 게임에 모두 담았고,
진행 상황은 자동 저장되어 **이어하기**가 가능합니다.

## 게임 방법

1. **자원 채집 (클릭형)** — 상단의 `나무 베기 / 식량 채집 / 금화 줍기` 버튼을 눌러 초기 자원을 모읍니다.
2. **건물 건설 (타일 배치형)** — 오른쪽 건설 메뉴에서 건물을 고르고, 격자의 빈 칸을 클릭해 배치합니다.
   - 🏠 **오두막** — 최대 인구를 늘립니다.
   - 🌱 **농장** — 식량을 자동 생산합니다. (일꾼 1 필요)
   - 🪓 **벌목장** — 나무를 자동 생산합니다. (일꾼 1 필요)
   - 🏪 **시장** — 금화를 자동 생산합니다. (일꾼 1 필요)
   - ⛏️ **철거** — 건물을 제거하고 비용의 50%를 돌려받습니다.
3. **자원 관리 (시뮬)** — 인구는 식량을 소비하고, 식량이 충분하면 (그리고 오두막으로 거주 공간이 있으면) 늘어납니다.
   생산 건물은 **일꾼(인구)** 이 있어야 제 성능을 냅니다. 일꾼이 모자라면 효율이 떨어지니
   오두막과 농장의 균형을 잘 맞춰야 합니다.

## 저장 / 이어하기

- 진행 상황은 브라우저 `localStorage`에 **자동 저장**됩니다. 새로고침해도 마을이 유지됩니다.
- `💾 저장` 버튼으로 즉시 저장할 수 있습니다.
- `🌱 새 게임` 버튼으로 처음부터 다시 시작합니다. (확인 후 현재 마을 삭제)

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```

## 구조

```
src/
  game/
    config.js     # 자원·건물 정의, 게임 상수
    engine.js     # 순수 게임 로직 (tick, 건설/철거, 통계 계산)
  hooks/
    useVillage.js # 상태 관리, tick 루프, 저장/불러오기
  components/
    ResourceBar.jsx   # 상단 자원/인구 표시
    BuildMenu.jsx     # 건설 메뉴
    VillageGrid.jsx   # 마을 격자
  App.jsx         # 전체 화면 조립
```
